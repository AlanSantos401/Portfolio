📊 DevBills — (Web + Mobile)

O DevBills é uma plataforma completa de gestão financeira disponível em versão Web e Mobile, desenvolvida para controle de finanças pessoais e empresariais com segurança e praticidade. O sistema possui integração total entre Web e App, compartilhando o mesmo backend e banco de dados em tempo real. Todas as informações cadastradas são sincronizadas simultaneamente entre as duas plataformas. A aplicação permite acompanhar receitas, despesas e saldo financeiro através de dashboards modernos e gráficos dinâmicos, oferecendo uma experiência intuitiva e responsiva.

🔹 Principais funcionalidades:

  • 🔐 Autenticação segura com Google Cloud
  • 🔄 Sincronização em tempo real entre Web e App
  • 📊 Dashboard com resumo financeiro atualizado
  • 📈 Gráficos dinâmicos com dados reais
  • 📝 Cadastro e gerenciamento de transações
  • 📱💻 Layout responsivo (Desktop, Tablet e Mobile)
  • ⚡ Atualização automática dos dados
  • 🧾 Controle completo de receitas e despesas

🛠️ Tecnologias utilizadas (Web):

  • React + Vite
  • TypeScript
  • Tailwind CSS
  • Axios
  • Google Cloud Auth

📱 Tecnologias utilizadas (App Mobile):

  • React Native
  • Expo
  • Expo Router / Navigation
  • TypeScript
  • Tailwind CSS
  • Axios
  • Google Cloud Auth

🧠 Destaques do Projeto:

  • ✔️ Backend único compartilhado entre Web e Mobile
  • ✔️ Dados sincronizados em tempo real
  • ✔️ Código organizado e escalável
  • ✔️ Interface moderna e intuitiva
  • ✔️ Arquitetura profissional

🌤️ DevTempo — (Web + Mobile)

O DevTempo é um projeto completo de previsão do tempo, disponível em versão Web e Mobile, desenvolvido com foco em usabilidade, visual moderno e dados climáticos em tempo real. O sistema permite ao usuário pesquisar qualquer cidade do mundo e visualizar informações meteorológicas de forma clara, intuitiva e responsiva.

🔹 Principais funcionalidades:

  • 🔍 Busca de cidades com navegação dinâmica
  • 🌡️ Clima atual (temperatura, sensação térmica, umidade e vento)
  • ⏰ Previsão horária para as próximas horas
  • 📅 Previsão estendida para os próximos dias
  • 🌙🎨 Background dinâmico que muda conforme o clima e período (dia/noite)
  • ⚠️ Tratamento de erros (cidade não encontrada, loading e estados vazios)
  • 🧼 Limpeza automática do input após a busca
  • 📱 Layout totalmente responsivo (Web, Tablets e Mobile)

🛠️ Tecnologias utilizadas (Web):

  • React
  • TypeScript
  • Styled-Components
  • OpenWeather API
  • Hooks personalizados
  • Componentização e boas práticas de estado
  • Git & GitHub

📱 Tecnologias utilizadas (App Mobile):

  • React Native
  • Expo
  • TypeScript
  • Expo Router
  • Axios
  • OpenWeather API
  • Hooks personalizados
  • Git & GitHub

📌 Projeto focado em:

  • ✔️ Boa experiência do usuário (UX)
  • ✔️ Código organizado e escalável
  • ✔️ Tratamento correto de estados (loading, sucesso e erro)
