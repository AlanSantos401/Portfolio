Dev Cine

O projeto "Dev Cine" é uma aplicação web desenvolvida em React...

Principais funcionalidades:

  • Exibição da previsão do tempo atual para uma cidade inserida pelo usuário.
  • Informações detalhadas, como temperatura, umidade e condições climáticas.
  • Atualização dinâmica com base na cidade fornecida.

Ferramentas utilizadas:

  • React: construção da interface.
  • Axios: requisições HTTP.
  • Vite: desenvolvimento rápido.
  • gh-pages: deploy no GitHub.
