🤖 Projetos de Automação Inteligente

Apresento alguns dos meus projetos de automação desenvolvidos com n8n, integrando websites, WhatsApp, Telegram e APIs externas, com foco em agentes inteligentes que otimizam processos e geram valor real para os usuários e empresas.

1️⃣ Automação com n8n

  • Obetivo: Integrar diferentes serviços para otimizar tarefas do dia a dia, trazendo praticidade, escalabilidade e redução de trabalho manual.

Funcionalidades:

  • 🌍 Envio e recebimento de mensagens automatizadas via website.
  • 📲 Integração com WhatsApp via Evolution API.
  • 🔈 Conversão de texto em áudio (TTS) e envio automático.
  • 📊 Criação de relatórios em tempo real e envio para canais de comunicação.
  • ⚡ Execução de tarefas em paralelo e orquestração de múltiplas APIs.

  • Tecnologias: n8n, Evolution API, Docker, APIs externas (TTS, análise de dados).

2️⃣ Agentes Inteligentes por Área

  • Agente de Curiosidades: Envia fatos, dicas e curiosidades para engajamento dos usuários.
  • Agente Financeiro: Analisa dados financeiros, gera relatórios e envia alertas.
  • Agente de Vendas: Automatiza follow-ups, envio de propostas e notificações para clientes.
  • Agente de Clínicas: Gerencia agendamento, envio de lembretes e relatórios de pacientes.
  • Análise de Reels: Coleta e analisa métricas de vídeos para otimização de conteúdo.

🐍 Jogo da Cobrinha

O projeto "Jogo da Cobrinha" é uma aplicação web desenvolvida com HTML, CSS e JavaScript, inspirada no clássico jogo dos celulares antigos. Criado com o objetivo de proporcionar diversão nos momentos de lazer, o projeto foi destaque e recebeu o prêmio "Baby Dev - Projeto do Mês (Abril)" no curso DevClub.

Principais funcionalidades:

  • 🎯 Controle do movimento da cobrinha via teclado e botões mobile.
  • 💾 Sistema de recorde com armazenamento local (localStorage).
  • 💥 Feedback visual ao perder, com mensagem de Game Over.
  • 🧮 Pontuação em tempo real durante o jogo.

Ferramentas utilizadas:

  • Html:5: Estruturação da interface do jogo.
  • Css: Estilização visual e responsividade.
  • JavaScript: Lógica do jogo, controle da cobrinha, geração de alimentos, detecção de colisões e interação com o usuário.

📖 Estudando com Fé

Plataforma interativa para estudo do cristianismo com apoio da Inteligência Artificial.
Estudando com Fé é uma plataforma voltada para o estudo do cristianismo, combinando tecnologia e fé.
O projeto conta com um Agente Cristão inteligente, que responde dúvidas e auxilia nos estudos bíblicos, além de oferecer um calendário devocional e um quiz interativo para testar seus conhecimentos.
A plataforma é totalmente responsiva, funcionando perfeitamente em celulares, tablets e desktops.


Principais funcionalidades:

  • 📖 Leitura da Palavra de Deus com fácil navegação.
  • 🤖 IA cristã para tirar dúvidas e aprofundar o conhecimento bíblico.
  • ❓ Quiz para testar e reforçar o aprendizado.
  • 📱 Layout responsivo adaptado para diferentes dispositivos.

Ferramentas utilizadas:

  • React + Vite: Interface moderna com alta performance e modularidade.
  • Tailwind CSS: Estilização eficiente com foco em responsividade e organização.
  • JavaScript: Lógica de aplicação, manipulação de dados e renderização dinâmica com React e APIs.
  • n8n: Automação e integração da IA com fluxos inteligentes e respostas automatizadas.

🎥 Dev Cine

DevCine é uma aplicação web interativa e moderna que permite aos usuários explorarem um catálogo de filmes, séries e animes.
A plataforma exibe pôsteres, sinopses e dados principais das produções, oferecendo uma navegação fluida e uma interface visual atrativa. Ideal para quem deseja descobrir novos conteúdos ou manter um catálogo visual das suas obras favoritas.

  • JavaScript: Lógica de aplicação, manipulação de dados e renderização dinâmica com React.
  • React Router (Route): Responsável pela navegação entre páginas como Home, Filmes, Series e Animes.

🍔 Prime Burguer

O Prime Burguer é uma aplicação web moderna e interativa desenvolvida para facilitar a experiência de navegação e pedido de hambúrgueres artesanais. A plataforma permite ao usuário explorar o cardápio, montar seu pedido, adicionar ao carrinho e simular a finalização da compra de forma rápida e intuitiva.

A aplicação é dividida em dois repositórios: um para a interface (front-end) e outro para a API (back-end), permitindo maior escalabilidade e organização do projeto.


Principais funcionalidades:

  • 🖼️ Listagem de hambúrgueres, acompanhamentos e bebidas com imagem, nome, descrição e preço.
  • 🗂️ Produtos divididos por categorias.
  • 🛒 Visualização dos itens selecionados com valor total dinâmico.
  • 📋 Interface clara com informações do pedido
  • ✏️🗑️ Cadastro, edição e exclusão de produtos.
  • 🔐 Estrutura pronta para autenticação de administrador (em desenvolvimento/expansão futura).

Ferramentas utilizadas:

  • React + Vite: Construção da interface com alta performance e modularidade no desenvolvimento.
  • Axios: Consumo de APIs externas para exibir dados de filmes, séries e animes.
  • Styled-components: Estilização dos componentes com foco em responsividade e organização do CSS.
  • JavaScript: Lógica de aplicação, manipulação de dados e renderização dinâmica com React.
  • Node.js + Express: Criação da API REST para gerenciamento de produtos.
  • SQLite: Banco de dados leve e eficiente para persistência local.

Repositórios:

Front-End: https://github.com/AlanSantos401/PrimeBurguer-FrontEnd.git
Back-End: https://github.com/AlanSantos401/PrimeBurguer-BackEnd.git

🌤️ Previsão do Tempo

O projeto Previsão do Tempo é uma aplicação web simples e intuitiva que exibe informações meteorológicas em tempo real de qualquer cidade do mundo. Utilizando integração com APIs de clima (como OpenWeatherMap), a aplicação permite ao usuário digitar o nome de uma cidade e receber dados atualizados.

Principais funcionalidades:

  • ⏱️ Previsão em tempo real.
  • 🌡️ Atualização automática da temperatura.
  • 🌦️ Condição do tempo (ensolarado, nublado, chuvoso, etc.).
  • 💧 Visualização de umidade.

Ferramentas utilizadas:

  • Html:5: Estruturação da interface.
  • Css: Estilização visual e responsividade.
  • JavaScript: Lógica de aplicação, manipulação de dados da API.
