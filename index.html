Jogo da Cobrinha

O projeto "Jogo da Cobrinha" é uma aplicação web desenvolvida com HTML, CSS e JavaScript, inspirada no clássico jogo dos celulares antigos. Criado com o objetivo de proporcionar diversão nos momentos de lazer, o projeto foi destaque e recebeu o prêmio "Baby Dev - Projeto do Mês (Abril)" no curso DevClub.

Principais funcionalidades:

  • Controle do movimento da cobrinha via teclado e botões mobile.
  • Sistema de recorde com armazenamento local (localStorage).
  • Feedback visual ao perder (mensagem de game over).
  • Pontuação em tempo real.

Ferramentas utilizadas:

  • Html:5: Estruturação da interface do jogo.
  • Css: Estilização visual e responsividade.
  • JavaScript: Lógica do jogo, controle da cobrinha, geração de alimentos, detecção de colisões e interação com o usuário.

Repositório:

Dev Cine

DevCine é uma aplicação web interativa e moderna que permite aos usuários explorarem um catálogo de filmes, séries e animes.
A plataforma exibe pôsteres, sinopses e dados principais das produções, oferecendo uma navegação fluida e uma interface visual atrativa. Ideal para quem deseja descobrir novos conteúdos ou manter um catálogo visual das suas obras favoritas.


Principais funcionalidades:

  • Busca de filmes, séries e animes por nome.
  • Atualização dinâmica dos dados via API.
  • Página inicial com destaques ou lançamentos.
  • Layout responsivo para desktop e mobile..

Ferramentas utilizadas:

  • React + Vite: Construção da interface com alta performance e modularidade no desenvolvimento.
  • Axios: Consumo de APIs externas para exibir dados de filmes, séries e animes.
  • Styled-components: Estilização dos componentes com foco em responsividade e organização do CSS.
  • JavaScript: Lógica de aplicação, manipulação de dados e renderização dinâmica com React.
  • React Router (Route): Responsável pela navegação entre páginas como Home, Filmes, Series e Animes.

Repositório:

https://github.com/AlanSantos401/DevCine.git

Prime Burguer

O Prime Burguer é uma aplicação web moderna e interativa desenvolvida para facilitar a experiência de navegação e pedido de hambúrgueres artesanais. A plataforma permite ao usuário explorar o cardápio, montar seu pedido, adicionar ao carrinho e simular a finalização da compra de forma rápida e intuitiva.

A aplicação é dividida em dois repositórios: um para a interface (front-end) e outro para a API (back-end), permitindo maior escalabilidade e organização do projeto.


Principais funcionalidades:

  • Listagem de hambúrgueres, acompanhamentos e bebidas com imagem, nome, descrição e preço.
  • Produtos divididos por categorias.
  • Visualização dos itens selecionados com valor total dinâmico.
  • Interface clara com informações do pedido
  • Cadastro, edição e exclusão de produtos.
  • Estrutura pronta para autenticação de administrador (em desenvolvimento/expansão futura).

Ferramentas utilizadas:

  • React + Vite: Construção da interface com alta performance e modularidade no desenvolvimento.
  • Axios: Consumo de APIs externas para exibir dados de filmes, séries e animes.
  • Styled-components: Estilização dos componentes com foco em responsividade e organização do CSS.
  • JavaScript: Lógica de aplicação, manipulação de dados e renderização dinâmica com React.
  • Node.js + Express: Criação da API REST para gerenciamento de produtos.
  • SQLite: Banco de dados leve e eficiente para persistência local.

Repositórios:

Front-End: https://github.com/AlanSantos401/PrimeBurguer-FrontEnd.git
Back-End: https://github.com/AlanSantos401/PrimeBurguer-BackEnd.git

Conversor de Moedas

O Conversor de Moedas é uma aplicação web desenvolvida para facilitar a conversão entre diferentes moedas de forma rápida e prática. Com uma interface simples e responsiva, o projeto permite que o usuário selecione moedas de origem e destino e visualize o valor convertido com base nas taxas de câmbio mais recentes.

Principais funcionalidades:

  • Conversão em tempo real ao digitar um valor.
  • Atualização automática com a taxa de câmbio atual.
  • Seleção de moeda de origem e destino via dropdown.
  • Interface amigável e compatível com dispositivos móveis.

Ferramentas utilizadas:

  • Html:5: Estruturação da interface do conversor.
  • Css: Estilização visual e responsividade.
  • JavaScript: Lógica de aplicação, manipulação de dados e renderização dinâmica.
